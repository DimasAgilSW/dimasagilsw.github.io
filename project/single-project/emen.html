Project DASW - EMEN Special Cream For Man

Project Year - 2019

This website is the official website of the St. Albertus de Trapani church which is located at Blimbing, Malang City, East Java, Indonesia. I created this website as an information center for church activities and also contains some information such as church organizational structure, form data for church needs. In addition, the purpose of making this website is also to digitize media that can be used to provide the required information.

This website uses Worpress and has registered the .id domain.
